# F1 Pulse — Design Context

## Identity
Premium F1 data analytics platform. Dark-first, data-dense, racing aesthetic.
Must feel like a custom-built F1 analytics tool — NOT a generic dashboard template.

## Palette
- **Primary:** F1 Red `#e10600` (accent, CTAs, active states)
- **Background:** `#15151e` (darkest), `#1e1e2a` (surface), `#22222f` (card)
- **Text:** White (primary), `rgba(255,255,255,0.55)` (sub), `rgba(255,255,255,0.25)` (dim)
- **Podium:** Gold `#ffc906` (P1), Silver `rgba(255,255,255,0.8)` (P2), Bronze `#cd7f32` (P3)
- **Tire compounds:** Soft `#ff3333`, Medium `#ffc906`, Hard `#e8e8e8`, Inter `#39b54a`, Wet `#0067ff`
- **Team colors:** Per driver (Red Bull `#3671C6`, Mercedes `#27F4D2`, Ferrari `#E8002D`, McLaren `#FF8000`, etc.)
- **Rule:** 60% dark surfaces, 30% card/glass-card, 10% red accents

## Typography
- **Body/Display:** Titillium Web (F1 official font) — 300/400/600/700/900 weights
- **Mono/Data:** Fira Code — telemetry values, lap times, gaps, all numeric data
- **Data rule:** ALL numeric values use `font-variant-numeric: tabular-nums`
- **Heading hierarchy:** f1-2xl (32px/900) → f1-xl (24px/900) → f1-lg (18px/900) → f1-md (14px/700)
- **Section labels:** 10px/700, uppercase, 0.14em tracking, muted color

## Motion
- **Easing:** `cubic-bezier(0.16, 1, 0.3, 1)` (spring) for all animations
- **Timing:** 100ms micro, 300ms state, 500ms entrance, 800ms hero
- **Stagger:** 100-150ms between siblings
- **GPU-only:** transform + opacity only — never animate layout properties
- **NEVER:** bounce, elastic, or overshoot easing

## Anti-patterns (REJECT)
- Inter, Roboto, or any generic sans-serif
- Purple gradients or purple as a primary color
- Glassmorphism with heavy blur (subtle 20px on dark cards is OK)
- Nested card grids (cards inside cards)
- Bounce or elastic easing
- Gray text on dark backgrounds (use white with opacity instead)
- Generic dashboard templates that could be any SaaS product

## Component Patterns
- `.glass-card` — primary container with subtle backdrop-filter in dark mode
- `.f1-table` — condensed data table with team-colored borders
- `.pos-badge` — position indicator (gold P1, silver P2, bronze P3)
- `.tire-badge` — compound indicator with compound color
- `.section-title` — 10px uppercase label
- `.racing-stripe` — left border accent using `--team-color`
- `.carbon-stripe` — diagonal texture (F1 nav pattern)

## Light Mode
Class-based via `html.dark` toggle. All CSS vars swap via `:root` / `html:not(.dark)`.
Light background: `#f3f3f4`, cards: `#ffffff`, text: `#1c1c25`.
